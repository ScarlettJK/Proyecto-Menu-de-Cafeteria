\# Reflexión sobre el uso de LLM en el desarrollo



\## ¿Qué tan preciso fue el LLM en el primer prompt? ¿Tuviste que corregir algo manualmente?



El LLM fue bastante preciso en el primer prompt. Generó correctamente la estructura del sitio y utilizó adecuadamente Bootstrap. Sin embargo, como el prompt no tenía muchos detalles o especificaciones, el resultado fue algo básico. Aun así, no fue necesario corregir errores manualmente, ya que el código funcionaba bien desde el inicio.



\## ¿Qué diferencia notaste entre un prompt vago y uno con restricciones claras?



Noté que mientras más específico era el prompt y más restricciones incluía, el resultado era mucho mejor. El LLM entendía con mayor claridad lo que se esperaba y generaba código más alineado a lo solicitado. En cambio, cuando el prompt era más general, el resultado tendía a ser más ambiguo y menos detallado.



\## ¿En qué casos preferirías escribir el código tú mismo en lugar de usar el LLM?



Preferiría escribir el código yo misma cuando se trate de detalles específicos, como la elección de colores, estilos particulares o elementos distintivos como imágenes o logos. También cuando quiero tener un control más preciso sobre el diseño o comportamiento del sitio.
